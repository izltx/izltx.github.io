Python-based Threat Intel & Static Analysis Engine

I engineered a custom Python security tool designed to safely ingest Portable Executable (PE) files using Read Binary mode without triggering execution. The engine performs deep static analysis, securely extracting Hashes (via hashlib) and PE Headers (via pefile) to uncover embedded IOCs.

To automate threat triage operations, the engine integrates directly with the VirusTotal and Shodan APIs using the requests library. This intelligent automation rapidly correlates identified hashes against global intelligence databases to accelerate the incident response lifecycle.

hash_extractor.py 
import hashlib

def extract_sha256(file_path):
    with open(file_path, "rb") as f:
        return hashlib.sha256(f.read()).hexdigest()

import requests

def query_virustotal(file_hash, api_key):
    url = f"https://www.virustotal.com/api/v3/files/{file_hash}"
    headers = {"x-apikey": api_key}
    return requests.get(url, headers=headers).json()
